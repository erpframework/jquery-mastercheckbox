Demo Master Checkbox

Both checkbox groups are wrapped in a fieldset.check-group element and are added to the plugin with a single statement. Any selector can be used, so long as it matches elements that wrap one or more entire groups of related checkboxes:

$('.check-group').masterCheckbox();

The master checkbox in Set 1 is wrapped in a p.master element. The master checkbox in Set 2 is a input.master element. The default ".master" selector can be overridden as follows, but it is useful to note that the selector works within the contect of the group wrapper, and can point directly to the master checkbox or any element that wraps it:

$('.check-group').masterCheckbox({master_sub_selector: '.my-master'});
